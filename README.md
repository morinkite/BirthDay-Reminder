# Morin Reminder
A program to remind the birthday of your contacts
# Reqirements
### Python 3
[Download Python](https://www.python.org/downloads/)
### [PyQt5](https://pypi.org/project/PyQt5/)
```
pip install PyQt5
```
### [JalaliDate](https://github.com/aziz/jalalidate)
```
pip install jalalidate
```
